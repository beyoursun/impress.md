# Impress Book

> 使用 `Impress Book` 来惊艳你的听众

[slide]

## Installation

```bash
# install global
npm install -g impressbook
```

[slide]

## Usage

```bash
# start a demo
impressbook

# start your presentation
impressbook presentation.md
```

[slide]

## Syntax

Use `[slide]` to split the sliders

```markdown
# Slider1

[slide] 

# Slider2

[slide] 

# Slider3
```
